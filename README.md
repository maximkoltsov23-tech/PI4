# Team Task Board WPF Demo

Team Task Board - демонстрационное WPF-приложение для варианта 1 лабораторной работы. Программа показывает мини-аналог Trello: пользователь работает с колонками и карточками задач, видит участников онлайн и сохраняет последнюю локальную копию доски.

## Возможности

- создание и удаление колонок;
- добавление, редактирование и удаление карточек задач;
- перемещение карточек между соседними колонками;
- панель демонстрационных пользователей онлайн;
- локальное сохранение состояния доски в JSON-кэш;
- MVVM-структура без бизнес-логики во View.

## Запуск

1. Откройте `PI4.slnx` или `PI4.csproj` в Visual Studio.
2. Убедитесь, что установлен .NET Framework 4.7.2 Developer Pack.
3. Соберите проект в конфигурации `Debug` или `Release`.
4. Запустите приложение `PI4`.

Также проект можно собрать из Developer PowerShell командой:

```powershell
dotnet build .\PI4.csproj
```

## Работа с программой

- Введите название в поле сверху и нажмите `Добавить колонку`.
- Введите название карточки внутри нужной колонки и нажмите `+`.
- Нажмите `Править`, чтобы открыть карточку в правой панели и изменить название, описание, исполнителя, приоритет или срок.
- Используйте стрелки на карточке, чтобы переместить ее в соседнюю колонку.
- Нажмите `Сохранить`, чтобы записать текущую доску в локальный JSON-кэш.

Кэш хранится в папке `%APPDATA%\PI4TeamTaskBoard\board-cache.json`.

## Поддерживаемая платформа

- Windows 10/11;
- .NET Framework 4.7.2;
- WPF.

Эта версия является локальным WPF demo. Настоящий SignalR-сервер, синхронизация между несколькими клиентами, CI/CD и кроссплатформенная Avalonia-версия в данный архив не входят.
