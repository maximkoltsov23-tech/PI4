# Отчёт по проекту Team Task Board

## Назначение

Приложение реализует демонстрационную совместную доску задач для варианта 1. Пользователь может управлять колонками и карточками, имитируя рабочий процесс команды: `К выполнению`, `В работе`, `Готово` и любые дополнительные этапы.

## Архитектура

Проект построен по MVVM:

- `Models` содержит POCO-модели доски, колонок, карточек и пользователей.
- `ViewModels` содержит `MainViewModel` с состоянием экрана, командами и правилами изменения доски.
- `Services` содержит `BoardStorageService`, который отвечает за чтение и запись JSON-кэша.
- `Infrastructure` содержит универсальную команду `RelayCommand`.
- `MainWindow.xaml` содержит только интерфейс и привязки WPF.

View не выполняет сетевые вызовы и не содержит бизнес-логику. Все действия пользователя проходят через команды ViewModel.

## Реализованный функционал

- создание и удаление колонок;
- добавление, редактирование и удаление карточек;
- перемещение карточек между колонками;
- отображение списка участников онлайн;
- сохранение и восстановление доски из локального JSON-файла.

## Распределение ролей

- Серверная часть: в demo-версии заменена локальным сервисом хранения состояния.
- Model и сервисы: модели данных и JSON-кэш.
- ViewModel: команды, изменение состояния, выбор карточки, перемещение задач.
- View и стилизация: WPF-разметка, стили, панель доски и деталей карточки.

## Ограничения demo-версии

Приложение не подключается к настоящему серверу и не синхронизирует данные между несколькими клиентами. Оно предназначено для локальной демонстрации интерфейса, MVVM-подхода и базовой логики варианта 1.

## Тестирование

Проверяются сценарии:

- сборка проекта без ошибок;
- добавление новой колонки;
- добавление карточки в колонку;
- редактирование карточки в правой панели;
- перемещение карточки влево и вправо;
- удаление карточки и колонки;
- перезапуск приложения с восстановлением данных из JSON-кэша.
